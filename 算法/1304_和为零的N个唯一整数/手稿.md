## 分析：

软柿子， 按n为奇数或偶数分类，然后从开始取值$[-\frac{n}{2},\frac{n}{2}]$即可，其中奇数取到0，偶数取不到0

***

 方法一：C++

```C++
class Solution
{
    public:
        vector<int> sumZero(int n)
        {
            vector<int> ret_val ;
            if(n >= 1)
            {
                /*奇数*/
                if(n % 2)
                {
                    ret_val.push_back(0);
                }
                
                for(int i = 1 ; i <= n / 2 ; i++)
                {
                    ret_val.push_back(-i);
                    ret_val.push_back(i); 
                }
            }
            return ret_val;
        }
};
```
